Ciao # Processed Data Directory

This directory contains preprocessed data files generated by `data_preprocessing.py`:

- `X_train.npy` - Training features
- `X_val.npy` - Validation features
- `X_test.npy` - Test features
- `y_train.npy` - Training labels
- `y_val.npy` - Validation labels
- `y_test.npy` - Test labels

These files are generated automatically when you run the preprocessing pipeline.
